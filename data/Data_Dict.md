
## Planet Rotation Curves

| Name       | Units | Description                                             |
|------------|-------|---------------------------------------------------------|
| name       | None  | The name of the planet                                  |
| R_dist     | km    | The semi-major axis of the planet's orbit about the Sun |
| Vobs       | km/s  | The average speed of the planet as it orbits            |
| perihelion | km    | The closest distance the planet gets to the Sun         |
| aphelion   | km    | The furthest distance the planet gets from the Sun      |
| errV       | km/s  | The error or uncertainty in the Vobs measurements       |
